**提问：FY**
> 看到牢马有新同事关注经济话题，提问：
> 最近房屋养老金是不是本质上的房地产税，大的要来了？！

**回答：由嘉由嘉**

谢谢你的提问！

任何强制购买的商品本质上都是一种税收，所以房屋养老金自然也是，但我在想的问题是，为什么现在一边各种政策楼市，却又一边“收税”压需求呢？如果想收税为什么不多从其他地方收再做转移？

税收分为两类，流量税和存量税

流量税主要是增值税和所得税征取，这是中国非常主流的种类，因为从利润中收取，阻力很小，大家赚得多自然缴多一点也无妨

但房产税，或者说房屋养老保险，是存量税。也就是说有房子但没收入也要缴税，阻力非常大

虽然房产税本质上对房价的影响可能不大，但确实可能会促使房价雪崩，这种情况下还要推行“房产税”，可能代表对未来“流量增长”抱有悲观态度

不过，税收考虑的问题肯定远比这个复杂很多，这个一个角度，仅供参考！

**提问：howe**
> 你好，嘉姐，很多人都说这是一个内卷的时代，潜台词是大家都很努力做到自己学习或者工作的“极致”。我有一个预设：“在市场上，激烈的竞争大概率带来更好的产品和服务”。
> 但是，我发现很多行业，比如装修行业、进出口贸易行业、心理咨询行业、医疗服务行业、高考志愿填报咨询等以服务业为主的行业，所谓的“内卷”，似乎并没有提供更让消费者更透明、更便利、更放心、更满意的服务，在价格激烈“内卷”的同时，尽管广告满天飞，消费者也许还要担心最基本的问题：装修公司不要“跑路”。
> 装修行业的内卷，似乎带来的是这个行业以低报价和必然的专业信息差来赚钱，不少服务业亦然。这个消费越低频和非标准化，越容易造成这种情况。
> 我本身的预设：“在市场上，激烈的竞争大概率带来更好的产品和服务”，是否并不成立呢？或者说，应当有什么条件才成立？
> 或者说，我的观察也有某种“有色眼镜”，出现了某种偏差，希望能够分析分析。

**回答：由嘉由嘉**

因为很多人没在卷“服务”，而是在卷“释放服务好的信号”

以什么标准竞争就会促进什么标准发展

原始森林的阳光资源按照身高分配，所以长出参天大树；现代教育资源按照学生考试分数分配，就，出了很多做题专家；市场经济按照需求解决的贡献分配，所以企业就会做出好的产品满足用户需求

但这里有一个问题，竞争标准无法透明，只能通过一些表象“信号”来判断，但“信号”和“能力”之间还有一层信息差

比如分配教育资源

教育体系的根本目的是希望学习能力强的人分配更优质的教育，充分发挥这些学生的才智

但你无法真的知道哪些学生学习能力强，你只能通过考试分数来判断，结果导致竞争变的畸形，变成了追求谁更会做题

市场也是一样，消费者探寻商家能力成本很高，只能去看商家释放的信号，以“广告”，“历史数据”，“背书认证”的方式来判断

结果就是部分商家单纯做营销，刷数据，释放“我能满足需求”的信号，特别是在你说的“低频交易”和“非标交易”领域，效果特别好，所以需要“监管”来提前筛选

但整体来说这个竞争的判断标准还是可用的，就好像应试教育确实培养了很多只知道做题的人，但也起到了筛选学习能力强的功能

而且，这套信号筛选体系也在不断进步，商业化在追求“标准”和“普及”，只要不干涉，信息也就会越来越透明

印钱可以稀释债务，通胀跑赢利息就意味着这笔钱的购买力变弱，债务也就是被稀释了

适当的通胀是正常现象。印钱是为了稳定商品的物价，因为市面上的商品在不断生产，如果没有足够多的钱，钱的购买力就在不断上升，结果就是没人愿意花钱了

举一个简单的例子，假如市场上有10辆车，定价一共100万，每辆车10万。但如果后续继续生产，有了20辆车，但没有新的钱，那车子的价格骤降，钱的购买力大幅上升，这样大家就不愿意消费，只愿意存钱了，因为每存一段时间，同样的钱就能买更多的商品，但没有消费也就不会有生产，形成恶性循环。

所以，适当的通胀是正常现象，为了对应生产出来的商品，需要印钱，而且要有一定程度上的通胀，让钱略微贬值来促进消费，促进流通和消费

但最关键的是你第三个问题，为什么有时候“印钱”也无法提高通胀

现代经济学对于通胀的理解框架是基于“货币数量论”，这个理论就是市面上流通的钱等于市场上所有商品的总价就是正常，略高于就有一定通胀刺激经济

但这个理论有一个问题

钱不花不是钱，货不用不是货，你根本无法通过观察市场上的钱和货来判断情况

没有需求，钱和货进不了市场流通，就是废纸和垃圾。随意，印钱不一定创造消费需求，生产也不一定创造社会财富，前者是经济滞胀，后者是产能过剩

精耕细作的模式形成了不同的生产制度

中国地区的地主对佃农采用的是固定利率地租，因为水稻生产极其复杂，地主不可能控制一切要素，所以，生产细节由农民自主决定，结果就是多劳多得。
